# 🎡 Бакыт Дөңгөлөгү (Колесо Удачи)

Интерактивная образовательная игра в формате «Поля чудес», направленная на изучение кыргызской культуры и языка. 

## 🚀 О проекте
Проект представляет собой Single Page Application (SPA), где игрок угадывает слова по буквам, вращая барабан. Темы слов охватывают географию, эпос и традиции Кыргызстана.

### Основные возможности:
* **Двуязычность:** Поддержка кыргызского и русского интерфейсов.
* **Интерактивный барабан:** Реализован на HTML5 Canvas с физикой вращения.
* **Звуковой движок:** Процедурная генерация звука через Web Audio API (без внешних аудио-файлов).
* **Система прогресса:** 3 раунда, подсчет очков и система жизней.

## 🛠 Технологический стек
* **Frontend:** React 18 (Hooks, Functional Components)
* **Стилизация:** Tailwind CSS
* **Графика:** HTML5 Canvas API
* **Анимации:** CSS Keyframes
* **Сборка:** Babel (Standalone)

## 📦 Как запустить
1. Склонируйте репозиторий: `git clone https://github.com/ваш-логин/bakyt-wheel-game.git`
2. Откройте файл `index.html` в любом браузере.
